# Meu Currículo Online

## Descrição
Este é um currículo online responsivo desenvolvido como trabalho prático para a disciplina de Desenvolvimento Web.

## Tecnologias Utilizadas
- **HTML5**: Estrutura semântica da página
- **CSS3**: Estilização, layout responsivo e animações
- **JavaScript**: Interatividade (smooth scroll, validações, animações)
- **PHP**: Processamento do formulário de contato

## Estrutura do Projeto
```
/meu-curriculo
├── index.html          # Página principal
├── style.css           # Estilos CSS
├── script.js           # JavaScript
├── contato.php         # Processador do formulário
├── img/                # Imagens (foto de perfil, etc.)
├── assets/             # Outros recursos
└── README.md
```
